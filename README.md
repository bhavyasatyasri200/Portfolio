# Karri Bhavya Satya Sri - Personal Portfolio

A sleek, modern, and high-performance personal portfolio website built from scratch with React, Vite, and Tailwind CSS v4.

## 🚀 Live Demo
https://bhavya-ten.vercel.app/

## ✨ Features
- **Responsive Layout:** Perfectly scales across mobile (375px), tablet (768px), and desktop (1280px) devices without horizontal scrollbars.
- **Dynamic Animations:** Features parallax scrolling and immersive scroll-triggered animations created with Framer Motion and React Scroll Parallax.
- **Accessibility Ready:** Includes `prefers-reduced-motion` to tone down animations based on user system preferences.
- **Performant:** Optimized for perfect Google Lighthouse scores.

## 💻 Tech Stack
- Frontend: React JS (Vite)
- Styling: Tailwind CSS v4
- Animations: Framer Motion & React Scroll Parallax
- Icons: Lucide React

## Run the Project locally
```
git clone https://github.com/bhavyasatyasri200/Portfolio.git
cd Portfolio
npm install
npm run dev
```
